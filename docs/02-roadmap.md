# מפת דרכים (אחרי אימות v0)

## v1 — העמקת הליבה
מניות/מדדים/קריפטו עם תנודתיות אמיתית, מצבי שוק (bull/bear/crash), הלוואות ואשראי עם ריבית אמיתית, ציון בריאות פיננסית, פרימיום ראשון (מסלולי חיים נוספים, קוסמטיקה).

## v2 — חברתיות + עסק
חברים, אתגרים שבועיים, לוח מובילים (עם הסכמת הורים לקטינים), התחלת עסק פרטי כמסלול משנה.

## v3 — B2B
חבילת בתי ספר, דשבורד למורים/הורים, תוכן חסות חינוכי, שותפויות עם משרד החינוך/עמותות פיננסיות.

## עקרון מנחה
כל שלב נפתח רק אחרי שהקודם הוכיח שימור (retention) ולמידה בפועל — לא לפי "היה נחמד שיהיה".
